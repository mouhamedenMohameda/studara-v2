# Projects quick start

## Mobile (`mobile/`)

```bash
cd mobile
npm install
npm run start
```

## Backend API (`backend/`)

```bash
cd backend
npm install
npm run env:init
npm run dev
```

## Admin dashboard (`admin/`)

```bash
cd admin
npm install
npm run dev
```

## Secondary API/service (`services/fastify-api/`)

```bash
cd services/fastify-api
npm install
npm run dev
```
